# SIH_Tourism_project
SafeTrek – Tourist Safety System

SafeTrek is a smart tourist safety platform that provides real-time tracking, digital identity, and emergency SOS support for travelers along with a rescue team command center dashboard.

🚀 Features
📝 Tourist registration & document scan (UI demo)
🔐 Blockchain-based digital identity (concept)
📍 Live GPS tracking & safety monitoring
🚨 One-click Emergency SOS system
🚁 Admin command center for rescue teams

📊 Emergency analytics & response dashboard

🛠 Tech Stack
HTML • CSS • JavaScript

🚀Future Enhancements
Real GPS & Maps API integration
AI-based risk prediction
Mobile app version
Cloud database integration
Real IoT wearable device support

🎯 Impact
SafeTrek can help:
Reduce tourist risks
Improve rescue response time
Enable smart emergency coordination
Increase traveler confidence
